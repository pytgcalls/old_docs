Exceptions#

All PyTgCalls API errors live inside the exceptions sub-package: pytgcalls.exceptions.

from pytgcalls.exceptions import GroupCallNotFound

try:
    ...
except GroupCallNotFound as e:
    ...

Missing Library Errors#

exception pytgcalls.exceptions.NodeJSNotInstalled#

Node.js isn’t installed, raised by start() or run()

NodeJS Errors#

exception pytgcalls.exceptions.NodeJSNotRunning#

NodeJS core not running, do start() before call these methods, raised by join_group_call(), leave_group_call(), change_volume_call(), change_stream(), pause_stream() and resume_stream()

Deprecation Errors#

exception pytgcalls.exceptions.TooOldNodeJSVersion#

Node.js version is too old, raised by start() or run()

exception pytgcalls.exceptions.TooOldPyrogramVersion#

Pyrogram version is too old, raised by start() or run()

exception pytgcalls.exceptions.TooOldTelethonVersion#

Telethon version is too old, raised by start() or run()

MtProto Errors#

exception pytgcalls.exceptions.InvalidMtProtoClient#

You set an invalid MtProto client, raised by PyTgCalls()

exception pytgcalls.exceptions.NoMtProtoClientSet#

An MtProto client not set to PyTgCalls, raised by join_group_call(), leave_group_call(), change_volume_call(), change_stream(), pause_stream() and resume_stream()

exception pytgcalls.exceptions.TelegramServerError#

Telegram Server is having some internal problems, raised by join_group_call()

PyTgCalls Errors#

exception pytgcalls.exceptions.PyTgCallsAlreadyRunning#

PyTgCalls client is already running, raised by start(),

exception pytgcalls.exceptions.GroupCallNotFound#

Group call not found, raised by get_active_call(), get_call()

exception pytgcalls.exceptions.NoActiveGroupCall#

No active group call found, raised by join_group_call(), leave_group_call(), change_volume_call(),

exception pytgcalls.exceptions.NotInGroupCallError#

The userbot there isn’t in a group call, raised by leave_group_call()

exception pytgcalls.exceptions.AlreadyJoinedError#

Already joined into group call, raised by join_group_call()

Stream Errors#

exception pytgcalls.exceptions.InvalidStreamMode#

The stream mode is invalid, raised by change_stream() or join_group_call()

exception pytgcalls.exceptions.NoVideoSourceFound#

This error is raised when the stream does not have video streams join_group_call() or change_stream()

exception pytgcalls.exceptions.InvalidVideoProportion#

FFmpeg have sent invalid video measure response, raised by join_group_call() or change_stream()

exception pytgcalls.exceptions.NoAudioSourceFound#

This error is raised when the stream does not have audio streams join_group_call() or change_stream()

CustomApi Errors#

exception pytgcalls.exceptions.TooManyCustomApiDecorators#

Too Many Custom Api Decorators, raised by on_update_custom_api(),

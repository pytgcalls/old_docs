Exceptions
==========
All PyTgCalls API errors live inside the ``exceptions`` sub-package: ``pytgcalls.exceptions``.

.. code-block:: python

    from pytgcalls.exceptions import GroupCallNotFound

    try:
        ...
    except GroupCallNotFound as e:
        ...
